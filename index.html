Free Game
  • Get a free game by clearing 4 or more in the same round of the main game.
  • Get 3, 5, 10, and 20 free games by clearing 4, 5, 6, and 7 consecutively in the same round.
  • In a free game, you can get 3, 5, 10, and 20 free games too by clearing 4, 5, 6, and 7 in the same round.
Bonus Game
  • In a main game, when three bonus icons appear on the win line, a bonus game can be triggered.
  • 1 stamina is deducted when fighter takes damage. The bonus game ends when stamina = 0.
  • Choose from 3 options: Dual swords, Single sword, or Lion.
  • Attack the lion or defend from lion's attacks to win rewards (lion can evade attack).
  • Payouts:
    • Dual swords – Bet × 20
    • Single sword – Bet × 10
    • Lion – Bet × 3
Wild

Wild represents any icon in the game (except bonus icon).

Game Rules
  • Bet line wins pay from left to right as per paytable.
  • Only the biggest payout is considered per payline.

  • Pay line:

  • Winning combinations follow the paytable rules.
Game Interruption Mechanism
  1. Malfunctions void all pays and plays. If a malfunction occurs during a feature game, the system will automatically complete the game and award the player.
  2. If a disconnection occurs after receiving the player's bet and the player can no longer influence the outcome, the bet result remains valid.
Payout Information

  • Maximum Payout: ‎600,000

  • Minimum bet: ‎1.2
  • Maximum bet: ‎1,200
  • Guaranteed 500× payout within 100,000,000 rounds of play.
Buttons
Spin
Tap to start spinning based on current bet. Can click reels or use spacebar (not available on all platforms).
Auto Spin
Tap to start auto spin or open settings:
  • Total Spins – Set how many rounds to play.
  • Stop if win exceeds set multiplier.
  • Stop if balance is less than or greater than set value.
  • Stop on Bonus Game or Free Game trigger.
